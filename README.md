## Установка

### Разработка

### Основной запуск

Склонируйте репозиторий:

```bash
git clone https://github.com/werserk/TechStormHack.git && cd TechStormHack
```

Запустите Docker:

```bash
docker compose up
```

### Запуск для разработки

Установите зависимости:

```bash
poetry install
```

Запуск стримлита:

```bash
streamlit run streamlit_app.py
```
